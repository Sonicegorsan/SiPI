Подробное описание

Класс, обрабатывающий соединение с клиентом и обрабатывающий запросы

Класс ClientHandler отвечает за обработку отдельного клиентского подключения. Он взаимодействует с базой данных, обрабатывает команды клиента и отправляет ответы. Поддерживает авторизацию, регистрацию, работу с сообщениями, задачами и отчетами.
