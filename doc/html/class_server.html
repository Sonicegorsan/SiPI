Подробное описание

Класс, реализующий TCP сервер для обработки подключений клиентов

Класс Server наследуется от QTcpServer и отвечает за прослушивание входящих соединений и создание обработчиков клиентов. Также класс инициализирует соединение с базой данных PostgreSQL.
