Подробное описание

Класс, реализующий окно авторизации и регистрации пользователей

LoginWindow предоставляет интерфейс для входа существующих пользователей и регистрации новых. Содержит поля для ввода имени пользователя и пароля, а также кнопки для выполнения авторизации и регистрации.
